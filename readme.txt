== Description ==

"inoday Web-to-Lead Zoho" will sync the lead or enquiry immidiate with Zoho CRM after contact form submission. You can review the log report at admin screen.

Major features included:

* Automatically checks all form fields and their value after form submission.
* Each of the enquiry will get synched with the Zoho CRM immidiate after form submission.

PS: You'll need an [Zoho AUTH TOKEN] to use it.  AUTH token you can generate from your Zoho CRM access URL.

== Installation ==

Upload the "inoday Web-to-Lead Zoho" plugin to your blog, Activate it, then enter your [Zoho AUTH TOKEN].

You need to use the input fields name like [your-name], [your-email], [your-phone] [your-message] only. We have mapped only four fields including the enquiry message in this version.

1, 2, 3: You're done!
